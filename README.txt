# Piano Responsive Project

This project is a responsive design exercise, focusing on creating a visual representation of a piano using HTML and CSS. This project is a guided exercise provided by freeCodeCamp.

## Features

- Detailed visual representation of a piano.
- Includes both white and black keys.
- Responsive design that adapts to different screen sizes.
- Includes the freeCodeCamp logo.

## Setup

1. Clone the repository to your local machine.
2. Open the `index.html` file in your preferred browser.

## Technologies Used

- HTML
- CSS

## Responsiveness

This project is designed to be fully responsive across multiple device sizes:

- Small devices (max-width: 768px)
- Medium devices (min-width: 769px and max-width: 1199px)
- Large devices (min-width: 1200px)
